This code defines a responsive, styled venue booking form with dark mode support. The form includes fields for entering the event name, purpose, number of attendees, and selecting a venue from predefined options. It also captures the start and end dates with corresponding time selections via dropdowns. Additionally, there is a textarea for specifying extra requirements such as technical needs or hosting details. The form concludes with a full-width submit button styled for prominence. All input fields are marked as required to ensure complete booking information. The layout uses utility classes for spacing, borders, and colors, adapting for light and dark themes.
