This code implements a responsive and styled venue booking form featuring dark mode compatibility. It includes input fields for event name, purpose, attendee count, and venue selection from preset options. The form captures start and end dates along with time selections through dropdown menus. A textarea is provided for additional requirements like technical or hosting needs. Consistent styling is applied across all inputs, and a full-width submit button labeled "Submit Booking" is included. The layout adjusts responsively, displaying date and time inputs side-by-side on medium and larger screens for improved usability.
